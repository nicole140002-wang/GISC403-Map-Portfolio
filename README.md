# GISC403-Map-Portfolio
This repository contains my GIS mapping portfolio developed for GISC403, showcasing thematic cartography and spatial visualization skills.
## Projects Included:
### 1. Canterbury Scenic Landscape Map
A base map of the Canterbury region displaying major rivers and road networks, with five selected scenic viewpoints highlighted as points of interest (POIs). This map demonstrates foundational cartographic composition, feature selection, and spatial labeling.
### 2. Africa Political Map
A political map of Africa showing national boundaries and regional geography. The design emphasizes clear cartographic hierarchy, boundary representation, and readable labeling for comparative regional visualization.
### 3. New Zealand Population Change (2023)
A thematic map visualizing population change across New Zealand regions in 2023. This project demonstrates basic spatial data visualization and interpretation of demographic patterns using choropleth mapping techniques.
## Tools & Skills
1) ArcGIS Pro
2) Cartographic design principles
3) Thematic mapping (choropleth, POI mapping)
4) Spatial data visualization
5) Vector data processing (roads, boundaries, points)
6) Information Extraction
